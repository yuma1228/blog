## 概要
Django で作ったブログアプリケーションです。
記事の投稿・閲覧・コメント機能を備えています。

## 主な機能
- ユーザー登録・ログイン / ログアウト
- 記事の作成・閲覧・削除
- コメントの投稿・削除
- プロフィール画像の設定
- マイページ（自分の投稿・コメント一覧、統計情報）

## 技術スタック
- Python / Django 6.0
- Bootstrap 5.3
- SQLite（開発） / PythonAnywhere（デプロイ）
- Pillow（画像処理）
